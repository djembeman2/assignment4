# Instructions  

## Attibutes

Attributes go in the opening tag of the HTML Element. They are used to change the behavior of the Element in a specific way. Each tag will have attributes specific to it. Check MDN or w3schools to know which attributes to use with which tags.

## Detailed instructions

3. Change the border of the table to make it look better.
4. Add an alt tag to the images. The alt tag will be used in place of the image in screen readers and should be descriptive for those with visual impairments.
