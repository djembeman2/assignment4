Attributes

Introduction

Attributes go in the opening tag of the HTML Element. They are used to change the behavior of the Element in a specific way. Each tag will have attributes specific to it. Check MDN or w3schools to know which attributes to use with which tags.

Abbreviations

The Abbr tag has an attribute called title. Check this link for details. Try hovering over an abbr tag with your mouse!

A webpage is written in HTML, which is the language that specifies how the page should look when rendered by a browser. Each webpage will have an address, or URL. An anchor tag is used to create a link to a webpage. Another name for this is a href, which is also the name of the anchor tag's attribute. Webpages are typically served over the protocols http or https.

Tables

Tables are for tabular data. They also specify a certain layout on the page. Tables have an attibute of border. Try changing the border attribute.
